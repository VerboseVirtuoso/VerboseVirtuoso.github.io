# Abhinav Vinod | Portfolio

A sleek, dark, futuristic, and highly interactive single-page portfolio website showcasing my skills, projects, certifications, and achievements as a Computer Science Engineer.

## Features
- **Dark & Futuristic Aesthetic:** Deep space colors (`#080b10`) accented with neon cyan (`#00d4ff`) and purple (`#7b2fff`).
- **Glassmorphism Design:** Frosted glass navbar and project cards providing visual depth.
- **Dynamic Interactions:** Custom neon cursor with hover tracing, and mouse-tracked radial gradients on all interactive cards.
- **Typewriter Effects:** Automated typing animation in the Hero section reflecting diverse engineering skills.
- **Scroll Reveal Animations:** Smooth fade-up animations on scroll for an engaging user experience.
- **Fully Responsive:** Adapts beautifully to mobile devices and tablets.
- **No Dependencies:** Built with pure vanilla HTML, CSS, and JS. 

## Projects Included
1. **Daycare Management & Operations Automation System:** Comprehensive hybrid-stack system built with PHP, Node.js, and MySQL.
2. **Online Group Study and Collaboration Platform:** A fully functional collaboration platform with shared study rooms built with HTML/CSS, JS, PHP, and MySQL.
3. **Django Quiz Management System:** A dynamic quiz application powered by a robust backend using Python and Django, featuring an aesthetic frontend.

## Installation & Setup

Since this is a static HTML file without external dependencies other than Google Fonts, running it is extremely simple:

1. Clone the repository or download the ZIP.
2. Navigate to the project directory.
3. Open `index.html` in any modern web browser.

## Technologies Used
- **HTML5**
- **CSS3** (Variables, Flexbox, CSS Grid, Keyframe Animations)
- **Vanilla JavaScript** (Mouse tracking, Custom cursors, Typing effects)
- **Google Fonts:** Space Mono & Syne
